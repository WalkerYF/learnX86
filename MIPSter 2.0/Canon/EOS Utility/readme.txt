EOS Utility软件的用途是设定您的佳能相机，并从相机中将图像传送出去。

本自述文件向您提供有关佳能实用程序EOS Utility的最新信息。
在使用EOS Utility之前，请一定通读本自述文件。

1. EOS Utility软件支持的文件格式
==================================
EOS Utility软件支持以下图像文件格式。
-使用佳能相机拍摄的图像文件
-符合DCF标准的JPEG图像

2. 有关连接佳能相机的说明
===================================
(1) 在EOS Utility与佳能数码相机之间进行数据传输的过程中，不要取出存储卡、卸下电池，或者断开通信电缆。根据您使用的计算机条件，这样做可能会引起故障。

(2) 当佳能相机连接到计算机上的USB端口或者IEEE1394端口上时，不要让计算机进入备用模式。当佳能相机连接到计算机上的USB端口或者IEEE1394端口上时，如果计算机因为某种原因进入了备用模式，请一定脱离备用模式，同时不要断开计算机上连接的电缆。在备用模式的过程中，如果电缆断开，可能无法脱离备用模式，这取决于计算机型号。有关备用模式的详细信息，请参阅您计算机的用户手册。

(3) 在多个软件程序同时运行时，相机不能进行通信。在计算机与相机之间进行数据传输的过程中，请一定保证只运行一个软件程序。

3. 有关装入图像的说明
=========================
当使用EOS Utility将图像装入您的计算机时，进行操作之前，请一定先检查图像的保存目标位置硬盘或磁盘上有足够的存储空间。如果您试图将图像装入计算机，而又没有足够的存储空间来保存这些图像，则图像的传输可能会中途停止，并且可能不能正确装入图像。

4. 有关在Windows XP下切换用户的说明
=========================
在使用EOS Capture（图像捕获）功能时，如果您切换用户，有时操作就不能正确进行。
