1. 安装 MovieEdit Task

MovieEdit Task 用于编辑用佳能相机拍摄的电影图像。它是从 ZoomBrowser EX 
启动的。
ZoomBrowser EX 版本 5.7 和更高版本支持 MovieEdit Task。
在安装 MovieEdit Task 前，请确保已安装 ZoomBrowser EX 版本 5.7 或更高版本。

如果您在 Windows 98SE、Windows Me 或 Windows 2000 上安装 MovieEdit Task，
那么可以使用 MovieEdit Task 和 MovieExport Task。
如果您在 Windows XP 上安装 MovieEdit Task，则可以使用 MovieEdit Task、
MovieExport Task 和 MovieFrame Task。

在 Windows 98SE、Windows Me 和 Windows 2000 上，MovieEdit Task 和 
MovieExport Task 将 QuickTime 用于内部处理。
这两个任务与 QuickTime 版本 5 和 6 兼容。
在启动 MovieEdit Task 或 MovieExport Task 前，确保 QuickTime 版本 5 或版本 
6 已安装在您的计算机上。
在安装 QuickTime 时，从安装选项中选择 [推荐]。


2. MovieEdit Task 不支持的图像

MovieEdit Task 和 Movie Export Task 不支持以下图像。
- RAW 图像
- 文件名超过 64 个字节的图像
- 大于 5,000 x 5,000 像素的图像
- 小于 50 x 50 像素的图像
- QuickTime VR 图像


3. 与处理能力较低的 PC 有关的说明

MovieEdit Task 要求极高的处理能力来处理电影图像。
在处理能力较低的计算机上预览图像时，可能会遇到动作不自然和声音不连贯的情况。
但是，实际文件本身不受这些问题的影响，并且将正常保存。
如果拼接预览中的动作，请尝试将 [任务环境设置] 中的 [预览大小] 设置为 
[4:3(320x240)]。

使用大量图像时，可用内存的数量将减少，并且您可能无法进行预览或保存。
在这种情况下，请减小图像大小或者减少添加到情节提要板中的图像数，然后重试。

此外，在无法使用音频设备的 PC 上，预览和保存操作可能无法正常运行。
在这种情况下，您应该使音频设备可用。


4. 保存图像

为了使用 MovieEdit Task、MovieExport Task 和 MovieFrame Task，
必须有足够的硬盘空间。
保存图像时，您应当确保用于保存图像的磁盘有足够的空间。
如果您试图将图像保存到空间不足的硬盘，就可能无法完成图像的保存，
或者可能无法正确保存它。
此外，如果图像较大或者帧速率较高，可能需要很长的时间才能保存。保存大小为 
640x480 或更大的图像时尤其要小心。

根据原始文件和保存图像类型之间的关系，声音质量可能会有所退化，
并且会产生比较明显的噪声。
如果发生这种情况，请在保存时更改声音类型的设置，然后再保存。


5. 与 Windows 98SE、Windows Me、Windows 2000 有关的说明

(1) 在安装 QuickTime 时，从安装选项中选择 [推荐]。
    选择 [最小] 将不会安装运行 MovieEdit Task 和 MovieExport Task 所必需的 
    QuickTime 组件。

(2) 如果您遇到与在 MovieEdit Task 中进行声音播放有关的问题，请尝试以下措施。
    - 从 [QuickTime 设置] 控制面板的弹出式菜单上选择 [输出声]。
      如果 DirectSound 可用，则检查是否已选取 [DirectSound]。
    - 单击 [选项] 按钮并增加 FIFO 大小（毫秒）。
    - 如果上面两种方法均无效，则在 [输出声] 设置中选择 [waveOut] 模式，
      并且增加缓冲区大小（毫秒）。
    - 如果您遇到与重播 8 位音频有关的问题，则在 [输出声] 设置中选择 [8 位]。

(3) 在 MovieEdit Task 中显示文本时，视 QuickTime 版本的不同，
    存在以下方面的局限性。
    - 在显示多行文本时，垂直显示位置可能稍微有些不对齐。
    - 如果在滚动至左侧时显示多行文本，则较长的文本可能显示为居中对齐。


6. 与 [临时保存位置] 有关的说明

(1) 如果在 MovieEdit Task 的 [任务环境设置] 中设置为 [临时保存位置] 
    的驱动器上的可用空间不足，则可能无法完成保存图像操作，
    或者无法正确保存图像。
    在这种情况下，请将 [临时保存位置] 更改为具有充足可用空间的驱动器。

(2) 如果您删除了在 [临时保存位置]（在 MovieEdit Task 的 [任务环境设置] 
    中所设置）中创建的临时文件，则可能无法预览该文件，或者无法正确保存该
    文件。
    请不要在运行 MovieEdit Task 时删除临时文件。
