1. 保存为 QuickTime VR 图像类型
-------------------------------

如果您安装了 QuickTime 3 或更高版本，则合并的图像可以 QuickTime VR 图像类型
保存。QuickTime 的最新版本可从 Apple Computer, Inc. 的网站下载。

http://www.apple.com/quicktime

注意：安装 QuickTime 软件时，
请选择包含所有文件的程序包。必须同时安装 QuickTime VR 程序和用于保存 
QuickTime 图像的程序，才能以 QuickTime VR 图像类型保存图像。


2. 在卸载 QuickTime 后与显示 QuickTime VR 图像有关的问题

--------------------------------------------------------

即使在 QuickTime 卸载后，仍可以 QuickTime VR 图像类型保存合并的图像文件，
因为 QuickTime 程序的一部分并未在卸载过程中删除。
但是，在试图查看保存的文件时会显示错误信息，因为所需的查看软件 MoviePlayer 
已在卸载过程中删除。
重新安装 QuickTime 后，可以查看用 QuickTime VR 图像类型保存的文件。


3. 与使用佳能数字打印机 CD-200 有关的打印问题的解决方法
-------------------------------------------------------

在从 Viewer 或 360Viewer 打印到佳能数字打印机 CD-200 时，可能会失去一部分图
像。要避免此问题发生，请在其中一个程序中单击 [文件] 菜单，选择 [选项]，然后

通过设置从 1 到 20 dpi 的数字，在打印前增加或降低打印分辨率。


4. 使用 Viewer 和 360Viewer 程序显示和处理大图像的相关建议

----------------------------------------------------------

在尝试从 PhotoStitch Launcher 或 PhotoStitch 程序显示大图像时，Viewer 或 
360Viewer 程序可能需要花一些时间来启动。在您尝试从 Viewer 或 360Viewer 程序

打印大图像时，打印对话框也可能会花一些时间来显示，并且其运行速度会非常慢。
在内存资源严重不足的系统上，该问题将会更突出。建议在具有充足内存资源的系统上
查看或打印大图像。


5. 合并使用远距转换器镜头或广角转换器镜头拍摄的图像

---------------------------------------------------

用相机上的远距转换器镜头或广角转换器镜头拍摄的图像无法在 PhotoStitch 中正确
合并。请避免使用这些镜头拍摄您要用 PhotoStitch 合并的图像。


6. Windows XP 和快速用户切换功能
--------------------------------

此程序已经过确认，可以在 Windows XP 系统下运行，但不支持 Windows XP 中的快速
用户切换功能。如果您在使用此程序时使用快速用户切换功能，则该功能可能不会正确
执行。要避免发生此问题，建议您在使用快速用户切换功能之前关闭该程序。


7. 无法合成以焦距低于 20mm 拍摄的图像（依据 35mm 胶片）
------------------------------------------------------

以低于 20mm 的焦距拍摄的图像（在 35mm 胶片使用中）无法在此软件中合成。在焦距

超过 20mm 的情况下拍摄。


8. 无法调用帮助文件
-------------------

如果无法调用帮助文件，请安装 Internet Explorer 4.01 SP1a 或更高版本。您可以
从 Microsoft 网站获得 Internet Explorer 的最新版本。


9. 保存为 QuickTime VR 图像类型时的压缩设置
-------------------------------------------

当您将图像保存为 QuickTime VR 图像类型，并在提供压缩设置的对话框中选择特定压
缩方法时，无法正常显示的图像有时也能保存。如果出现此情况，请另选一种压缩方法。
