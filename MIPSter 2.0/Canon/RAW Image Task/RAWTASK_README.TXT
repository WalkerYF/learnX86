1. 与运行 RAW Image Task 有关的说明

要运行 RAW Image Task，请首先在 ZoomBrowser EX 中选择 RAW Image 缩略图，
然后从 [任务区域] 的 [编辑] 下选择 [处理 RAW 图像...]。 您不能单独运行 
RAW Image Task。


2. 有关 [状态] 区域中显示的 RGB 值的说明

如果您设置的显示大小低于 100%，则 [状态] 区域中显示的 RGB 值是围绕该光标坐
标的区域的平均值。
若要显示特定坐标的正确 RGB 值，请将显示大小设置为 100% 或更高。
